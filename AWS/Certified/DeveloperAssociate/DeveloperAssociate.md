## AWS Certified Developer - Associate
AWS 认证开发者 - 助理级展示了在开发、优化、打包和部署应用程序、使用 CI/CD 工作流以及识别和解决应用问题方面的技能和知识。本认证是 IT 或云开发者岗位人员开始 AWS 认证之旅的良好起点。
