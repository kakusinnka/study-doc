# 用小部件填充屏幕
小部件是数字应用程序中屏幕（表单）的构建块，每个小部件都有特定的用途，例如用户交互或动画。 Volt MX Iris 为您提供内置小部件，帮助您实现所需的功能。您可以根据需要配置每个小部件。

## 从“小部件”选项卡访问小部件
这些小部件位于 Volt MX IrisLibrary 窗格的“小部件”选项卡上。要将小部件添加到表单中，只需将它们拖放到适当的位置即可。当您执行此操作时，将显示对齐参考线以指导您对小部件的定位。

## 三种类型的小部件
Volt MX Iris 提供三种类型的小部件：容器、基本和高级。

### Container Widgets  容器小部件
容器小部件充当对其他小部件进行分组的容器。使用容器小部件，您可以将两个或多个小部件分组，以便可以将它们定位为一个单元。以下是 Volt MX Iris 中可用的容器小部件：

* [FlexContainer](./FlexContainer.md): 具有灵活属性的容器小部件。灵活的属性允许您将小部件放置在窗体中的任何位置，并配置对齐属性，例如“左”、“右”、“上”和“下”。您可以用三种度量单位设置 FlexContainer 的属性：dp（与设备无关的像素，dip = dp）;px（屏幕上的实际图片元素 [即像素] ）;和百分比。如果需要，可以将 FlexContainer 转换为 FlexScrollContainer。
* 一个容器小部件，可用作 [FlexContainer](./FlexScrollContainer.md)，但可以水平或垂直滚动。
* [TabPane](./TabPane.md) 一个容器小组件，您可以在其中对小组件进行分组。每个选项卡都可以有自己的一组小组件，您可以使用选项卡小组件添加新选项卡。
* [Tab](./Tab.md)仅与 TabPane 小组件一起使用的容器小组件。将此小组件拖放到 TabPane 小组件上以创建新选项卡。

### Basic Widgets 基本小部件
基本小组件可帮助您在应用程序中构建用户界面。除了设计应用程序外，您还可以为这些小组件配置事件或编写代码片段以实现功能。

* [Button 按钮](./Button.md)：使用按钮小部件可以向应用程序提供输入或触发事件。例如，您可以导航到窗体、与对话框交互或确认操作。
* [Calendar 日历](./Calendar.md)：日历小部件允许您从图形日历中选择日期。“日历小组件”显示为带有小日历图标的标签（图标在移动 Web 平台上不显示），并显示您指定的日期或日期格式。单击“日历小组件”进行交互。
* 当用户可以进行一个或多个选择时，使用 [CheckBoxGroup](./CheckBoxGroup.md) 小组件从一组复选框中进行选择。复选框内的复选标记指示所选内容。
* DataGrid 数据网格: [DataGrid Widget](./DataGrid.md) 允许您以行和列（表格格式）显示数据集合。
* [Image2](./Image2.md): “图像”小组件是一个非交互式小组件，可用于显示图形（本地或远程）。您可以在显示公司徽标、显示快照和提供插图等场景中使用图像微件
* “[Label 标签](./Label.md)小组件”在表单上显示不可编辑的文本，并且是非交互式的。
* [列表框小组件](./ListBox.md)显示项目列表（如下拉框），并允许您一次选择一个项目。
* [RadioButtonGroup](./RadioButtonGroup.md) 是一个小部件，允许您定义一组单选按钮，用户可以选择其中一个按钮作为选项。
* [RichText](./RichText.md) 小部件在表单上显示不可编辑的格式化文本。RichText Widget 中的 HTML 格式标记，以显示带有样式（粗体、下划线等）、链接和图像的文本。
* [Slider](./Slider.md) 滑块小部件允许您通过在水平方向上移动拇指（指示器）从定义的值范围内选择一个值。
* [TextArea2](./TextArea2.md) Widget 提供了一种用户可以输入文本的方法。
* [TextBox2](./TextBox2.md) Widget 用于捕获用户的输入。

### Advanced Widgets 高级小部件
高级小部件使您能够实现应用程序中最常用的功能。这些小部件由 Volt MX Iris 开发。此外，还提供了配置小部件属性的选项。

* 使用 [Browser](./Browser.md) 浏览器小组件显示应用程序的 HTML 内容，而无需离开应用程序或打开本机浏览器。
* 使用 [Cordova](./Cordova.md) 浏览器小组件使用户可以访问 Cordova 应用程序的内容。有关详细信息，请参阅创建 Cordova 应用程序。
* [Camera](./Camera.md) 照相机: 使用设备的本机摄像头及其功能进行图像和视频捕获。
* [Map](./Map.md): “地图微件”在屏幕上的地图上显示由纬度和经度定义的位置。
* [Phone](./Phone.md) 电话: 访问本机电话拨号器，并对小组件上显示的号码发起语音呼叫。
* [PickerView](./PickerView.md) 小组件使用旋转轮控件或平视图选取器来显示多组值，并允许您选择值的组合。
* [Segment2](./Segment2.md) 小组件由多个段（行或记录）组成，每个段（行或记录）可以有多个子小组件。
* [Switch](./Switch.md) Widget 提供两种互斥的选择或状态。
* [Video](./Video.md) 视频：用于显示 URL 引用的视频或视频用户上传。

## 使用小部件填充表单
使用以下选项，您可以使用小部件填充表单：
* VoltMX IrisLibrary：您可以通过从小部件库中拖动小部件并将小部件拖放到表单上来使用小部件填充表单。
* 用户定义的集合：从用户定义的集合中，您可以拖放小组件。
* 复制(Duplicate)：如果您有要在同一表单中复制的小部件，请使用“复制”功能。要复制小组件，请右键单击小组件，然后单击复制。复制的小部件与前一个小部件的偏移量相差 10 个单位的水平和垂直偏移量，因此很容易与之前的小部件区分开来。
* 复制和粘贴：如果要在其他表单或频道上重复使用小组件，可以复制和粘贴它。若要复制和粘贴小组件，请右键单击小组件，单击“复制”，然后单击“粘贴”。
* 注意：复制(Duplicate)功能和复制和粘贴(Copy and Paste)功能的不同之处在于，您可以在同一表单中复制(Duplicate)小部件，而您可以复制(Copy)小部件，然后将小部件(Paste)粘贴到同一表单、不同表单或不同的通道中。

## 占用父构件的整个区域
您可以将小组件适合其父组件，使其占据父组件的整个区域。

在 Iris Canvas 上，右键单击要适合其父级的构件，然后单击“适合父级”。小组件的 Top 和 Left 属性设置为 0，Height 和 Width 属性设置为 100%，导致小组件占据父小组件的整个区域。

## 调整大小和重新定位小部件
您可以通过两种方式单独或作为多项选择的一部分调整微件的大小和重新定位：拖动其边界控点，或更改位于“属性”窗格的“外观”选项卡上的 Flex 属性。

1. 选择一个或多个小组件。有关详细信息，请参阅[选择多个项目](./SelectMultipleItems.md)。
2. 若要重新定位，请将鼠标悬停在小组件上（如果选择了多个小组件，则为任何一个小组件），然后将小组件拖动到所需位置。或者，在“属性”窗格的“外观”选项卡上，更改“左”或“右”以及“顶部”或“底部”的“Flex”属性。
3. 若要调整大小，请将鼠标悬停在边界控点上，然后将小组件拖动到所需大小。根据您是水平拖动还是垂直拖动，小组件相对于其原始尺寸以一个单位的增量重新调整大小。或者，如果希望构件具有相同的尺寸，请在“属性”窗格的“外观”选项卡上，更改“宽度”和“高度”的“弯曲”属性。
4. 在 Iris Canvas 上，单击其中一个选定小组件的上下文菜单箭头，指向 Group Into Flex，然后选择 Flex 容器（用于不可滚动的容器）或 Flex 滚动容器（用于可以垂直和水平滚动的容器）。

“属性”窗格的“外观”选项卡上的 Flex 属性如下所示：
* Left 左 - 确定小组件的左边缘，并从父容器的左侧边界进行测量。
* Top 顶部 - 确定小组件的顶部边缘，并从父容器的顶部边界进行测量。
* Width 宽度 - 确定小组件的宽度并沿 x 轴测量。
* Min Width 最小宽度 - 指定小组件的最小宽度。仅当未指定 width 属性时，才会考虑此属性。
* Min Height 最小高度 - 指定小组件的最小高度。仅当未指定 height 属性时，才会考虑此属性。
* Center X 中心 X - 确定从父容器的左侧边界测量的小组件的中心。
* Z Index Z 索引 - 指定小组件的堆栈顺序。zIndex 较高的小部件位于 zIndex 较低的小部件前面。
* Right 右 - 确定小组件的右边缘，并从父容器的右边界进行测量。
* Bottom 底部 - 确定小组件的底部边缘，并从父容器的底部边界进行测量。
* Height 高度 - 确定小组件的高度，并沿 y 轴（父项的高度）进行测量
* Max Width 最大宽度 - 指定小组件的最大宽度。仅当未指定 width 属性时，才会考虑此属性。
* Max Height 最大高度 - 指定小组件的最小高度。仅当未指定 height 属性时，才会考虑此属性。
* Center Y 中心 Y - 确定从父容器的顶部边界测量的小组件的中心。

## 将 Widget 分组到容器 Widget 中
使用 Volt MX Iris，您可以将小部件组合到单个容器小部件下，以便您可以将它们作为一个单元进行操作，包括移动和复制它们。

1. 选择要组合在一起的小组件。
2. 在 Iris Canvas 上，单击其中一个选定小组件的上下文菜单箭头，指向 Group Into Flex，然后选择 Flex 容器（用于不可滚动的容器）或 Flex 滚动容器（用于可以垂直和水平滚动的容器）。

## 复制小组件路径
如果您需要引用模块中的特定小部件，您可以复制小部件路径并将其粘贴到代码中，而不必键入它。小组件路径遵循它们所在的窗体和容器的层次结构。通用的基本小组件路径按以下方式组织：`FormName.ContainerName.WidgetName`

要复制小组件的路径，请执行以下操作：
1. 在“项目资源管理器”的“项目”选项卡上，导航到要复制其路径的小组件。
2. 单击小组件的上下文菜单箭头，然后单击“复制小组件路径”。

## “父项”或“取消父项”构件
您可以将小部件制作到父项中，然后在其下添加小部件，以便将它们组合在一起。

要父构件或非父构件，请执行以下操作：
1. 将小部件拖放到窗体上。
2. 从 Project Explorer 中选择小组件。
3. 从“项目资源管理器”中，将所需的小部件拖放到表单上。在 Project Explorer 上，您删除的小组件缩进到先前选择的小组件下。
4. 要取消小组件的父级，请右键单击子小组件，然后单击“取消父级”。小组件的链接将从父小组件中删除，并在项目层次结构中向上一级链接。

## 设置默认小组件样式
对于每种类型的小部件，Volt MX Iris 都有自己的通用默认样式，但您可能希望设置自己的默认小部件样式，以简化格式化新小部件的过程。

要设置默认小组件样式，请执行以下操作：
1. 在 Iris Canvas 上，选择要用作您创建的该类型的任何新构件的默认样式的小组件。
2. 单击小组件的上下文菜单箭头，然后选择“设置默认小组件样式”。您创建的该类型的任何新小部件都会自动使用新特征进行格式化。
3. 要根据 Volt MX Iris 对该小组件类型的默认格式重置要格式化的小组件类型，请单击该小组件的上下文菜单箭头，然后选择重置为 Iris 默认值。
