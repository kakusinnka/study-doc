# 实验简介：警告在 Google Cloud
在本实验中，您将向 GCP 部署应用程序，然后创建警报策略，以便在应用程序未启动或生成错误时通知您。

# [概述](https://www.cloudskillsboost.google/course_sessions/4808300/labs/410247)
在本实验中，您将应用程序部署到 App Engine，然后创建警报策略，以便在应用程序无法访问或生成错误时通知您。

# 目标
在本实验中，您将学习如何执行以下任务：
* 从 GitHub 下载示例应用程序。
* 将应用程序部署到 App Engine。
* 创建正常运行时间检查和警报。
* 或者，使用 CLI 创建警报策略。
