# Cloud Composer
基于 Apache Airflow 构建的全代管式工作流编排服务。

## 快速入门
创建环境，创建 DAG，将 DAG 上传到 Cloud Storage，在 Airflow 界面中查看 DAG，在 Airflow 日志中查看任务实例详情。
