# GET START
## 介绍视频
通过这组介绍性视频开始使用 Visual Studio Code 的旅程！ 这些视频旨在让您了解 VS Code 的各种功能并快速熟悉它们。
* Visual Studio Code 入门。在本教程中，我们将引导您完成 Visual Studio Code 的设置并概述基本功能。
* Visual Studio Code 中的代码编辑。在本 Visual Studio Code 教程中，我们将介绍如何编辑和运行一小段代码。
* 生产力提示。描述了有助于提高编码效率的 Visual Studio Code 功能。 学习一些技巧可以加快您的工作流程并帮助您发现 VS Code 编辑器的全部功能。
* 个性化 Visual Studio Code。将向您展示如何使用主题个性化 Visual Studio Code。
* 在 Visual Studio Code 中使用扩展。将向您展示如何查找扩展、安装您喜欢的扩展以及禁用您不想一直使用的扩展。
* 在 Visual Studio Code 中调试。将向您展示如何在 VS Code 中运行和调试程序。我们将浏览“运行和调试”视图，探索一些调试功能，最后设置断点。
* VS Code 中的版本控制。将学习如何在 Visual Studio Code 中使用 Git 版本控制的基础知识。
* 自定义 Visual Studio Code。将向您展示如何使用设置和键盘快捷键自定义 Visual Studio Code。
