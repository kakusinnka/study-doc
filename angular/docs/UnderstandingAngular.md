* Templates 模板
* Directives 指令
* Dependency injection 依赖注入

# [Components 组件](./Components.md)
组件是 Angular 应用程序的主要构建块。每个组件包括：
* 声明页面上呈现内容的 HTML 模板
* 定义行为的 TypeScript 类
* 定义如何在模板中使用组件的 CSS 选择器
* （可选）应用于模板的 CSS 样式
